# Eventful Backend

Eventful is a ticketing and event management backend built with Node.js (TypeScript), Express, and Prisma.

## Features

- User Authentication (JWT)
- Role-Based Authorization (CREATOR / EVENTEE)
- Event Creation (Creator only)
- Public Event Listing
- Ticket Purchase
- QR Code Generation for Tickets
- Ticket Scanning
- Payment Initialization (Paystack Integration)
- Event Analytics (Total, Scanned, Pending Tickets)
- Rate Limiting
- Swagger API Documentation

---

## Tech Stack

- Node.js
- TypeScript
- Express
- Prisma ORM
- PostgreSQL
- Paystack API
- Swagger (OpenAPI)
